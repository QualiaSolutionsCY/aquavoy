## First: get the repo running locally

```bash
git pull origin main
npm install
npx vercel link                  # qualiasolutionscy → aquavoy
npx vercel env pull .env.local   # SESSION_SECRET / OPERATOR_CREDENTIALS / ENCRYPTION_KEY etc.
npm run build && npm test        # build + 59 vitest tests green
npm run test:e2e                 # Playwright e2e (auth-gate + authed + 375px)
npm run dev
```

⚠️ **`ENCRYPTION_KEY` is load-bearing** — mailbox passwords + OAuth tokens are
AES-256-GCM encrypted under it. Don't change it; `vercel env pull` gives the live one.

Login operators are **Wency** / **Jeanette** (passwords hashed in prod
`OPERATOR_CREDENTIALS`). Note: `.env.local` ships **empty** `SESSION_SECRET=""` /
`OPERATOR_CREDENTIALS=""` placeholders — the real dev values live in
`.env.development.local` (Next.js loads it with higher priority).
