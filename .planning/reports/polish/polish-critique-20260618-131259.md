## Weakest dimension per route

- **Login** — Color/Type/Motion strong; **sparse on desktop** (single centered card in a large void). Minor: nav links to gated pages render on the login screen.
- **Chat** — Strong trace rows + bubbles; **wide single-column void at 1440**; token/latency numbers lack tabular alignment (Typography).
- **Emails** — Best page: semantic Connected badges + sent/cancelled status pills + mono metadata + 7 real `<label>`s. Lift: tabular nums on counts/times; row spacing slightly loose.
- **Files** — **error + empty double-state coexist** ("Could not load — Retry" banner above "This folder is empty") which reads confusingly; **two teal primaries** compete (`+ Connect account` and `Upload`); search input placeholder-only; huge empty panel.
- **Prep** — **vast empty right panel** on desktop; **placeholder-only inputs** (Name/Email/Role/Context — `prep/page.tsx:268-287`, against product-register "visible labels"); the remove control is a **red "X"** (alarming + small tap target).

## Top 5 fixes (priority order — drives the next `/qualia-polish`)

1. **Lift Visual system 2→3+ (unblocks the FAIL).** Add a glanceable layer: file-type icons in Files, the 12 mailboxes as a status grid/health strip on Emails, and a compact "today" summary (queue + recent activity) on Chat or a small dashboard. Rich ops data → make it *seen*, not just listed.
2. **Tabular numerals (Typography 3→4).** One rule — `font-feature-settings:"tnum" 1` on numeric/mono surfaces (traces, counts, times). Instantly more instrument-panel; near-zero risk.
3. **Tint the shadow + add one ambient level (Depth 3→4, closes §10 drift).** `oklch(0 0 0 / 0.3)` → hue-220-tinted; add a subtle ambient elevation token.
4. **Right-size empty states & desktop density (Spatial/Layout 3→4).** Shrink the oversized Files/Prep empty panels into centered icon + sentence + action; give Prep's empty right panel a useful default (recent drafts / tips) so it isn't a void.
5. **Input labels + control polish (product-register).** Visible labels on Prep/Files inputs; make Prep's remove a neutral icon button (≥44px, not red); show *one* of Files' error/empty states; demote one of the two teal primaries.

## What's already excellent (preserve — do NOT redesign)

OKLCH token discipline (0 hex), the committed teal-on-ocean identity, the sonar-sweep skeleton + bubble-in motion, reduced-motion support, the agent's microcopy voice, and Emails' semantic badge/status system. The maritime-console direction is working — this is refinement, not a pivot.
