Generated by `scripts/build-invoice-templates.ts` via `pizzip` + OOXML.

## Regenerating templates

```bash
npx tsx scripts/build-invoice-templates.ts
```

The script writes both `.docx` files and performs a docxtemplater render-test with
sample data before exiting. If any `{tag}` is malformed, the script throws before
writing (fail-fast).

## Engine

- `docxtemplater` `^3.69.0` — template fill (single-brace `{tag}` delimiters, `paragraphLoop: true`, `linebreaks: true`)
- `pizzip` `^3.2.0` — OOXML zip assembly / reading
- OOXML minimal document: `[Content_Types].xml`, `_rels/.rels`, `word/document.xml`, `word/_rels/document.xml.rels`
